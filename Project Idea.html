Overview

The project will be an interactive learning experience that will teach users to code in Python. The project is specifically geared towards network engineers, admins, technicians, managers, all looking to learn python and apply it to complex networking. The project aims to educate those already working, or looking to work, in networking. Python is extremely user friendly when compared to other coding languages and makes the experience of learning to code in Python not as scary.

Motivation

In networking, learning to code in Python will be crucial to staying relevant and up to date with modern networking requirements. From personal experience, the senior engineers in my workplace are all learning to code in Python and having this website as a resource that would be tailored to their needs would aid in their learning and understanding. In my workplace, Python is being utilized at a basic level and is used for compiling reports using keywords and terms, which will be used in a database that will be able to automate level 1 technical requests. If successful, level 1 requests could be resolved via automation or the troubleshooting of these requests could already be completed, with the help of Python.

Description

This project would operate in a similar fashion to another website run by Code Academy. The main difference is that only 1 programming language is taught, and it is taught specifically around networking. There will be 5 levels of learning, Basic, Beginner, Intermediate, Advanced and Expert.

A user is presented with a page where they can enter in their details, which includes their current level of experience with coding, what they do for work, how they’ll use what they’re learning in the real world, amongst other options. This information is collected and then translated to formulate a course structure for that student. To avoid too many variables, there will only be drop down menus or buttons to input information.

After the information has been submitted for the student, they’ll be presented an estimated completion date, goal tracker, etc, that will hopefully enable the student to complete the course with positive validation throughout.

To avoid drowning each student with hordes of information and becoming disinterested, lessons will be taught in 15-minute blocks. This is to ensure the student remains focused on completing the block and doesn’t have to spend too much time every day studying.

Upon completion of the course, a detailed transcript on what was taught and the skills each student has achieved will be presented. Vouchers for discounted certification tests will be present upon completion of the course, so the student can become certified.

Tools and Technologies

Python can be used in web development and would be the choice of code used to build this project. This project would be hosted with a hosting provider that has a global footprint and a global support network, as users would ideally be able to access the learning tool from anywhere in the world. The project would be fully hosted online, and used a cloud provider, such as AWS, to host the project. This allows for global access and scalability for future growth.

Skills Required

  • Python Web Development: Having a web developer who is able to code proficiently in python is absolutely crucial to this project.
  • Python Network Engineer: Having a skilled network engineer who has already learnt to code in python and apply it to networking scenarios is also crucial

    • These skills are all readily available. Some of the most used websites in the world have all been written with Python (Google, Reddit, Facebook). Finding a network engineer who is proficient in Python already would be difficult, but not impossible. During the process to find someone to perform the work, portfolios or examples of previous work would be required.

    Outcome

    The goal of this project is to teach network engineers how to code in Python. These engineers are generally already very skilled in networking and having the ability to transfer that skill and knowledge into python would greatly benefit themselves and their employers. If successful there will be a new generation of networking professionals that are skilled in both Python and Networking.
